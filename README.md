# Mini Rede Social Rails + React

## Instalação

1. `git clone git@github.com:renanvy/mini_social_network.git`
2. `cd mini_social_network`
3. `bundle install`
4. `rake db:create`
5. `rake db:migrate`
6. `rake db:seed`
7. `rails s`
8. `acesse localhost:3000`

## Rodar os testes

`rspec spec/`

## Funcionalidades

- Publicar post
- Editar post
- Remover post
- Comentar post
- Editar comentário do post
- Excluir comentário do post
- Curtir post
